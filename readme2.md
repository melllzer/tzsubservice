# Docker-развёртывание (опционально)

Если вы хотите запустить проект через Docker — пожалуйста, используйте следующую инструкцию.

>  Основной способ запуска описан в [README.md](README.md) (без Docker).  
> Этот вариант предназначен для тех, кто предпочитает контейнеризацию.

##  Требования

- Docker
- Docker Compose

##  Запуск

1. Убедиться, что в корень проекта добавлены:
   - `Dockerfile`
   - `docker-compose.yml`

2. Выполнить в терминале:

```bash
docker-compose up --build
